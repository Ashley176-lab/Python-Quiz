# Fun IT Quiz (Python Multi-Quiz System)

A simple interactive Python quiz application that allows users to test their knowledge in different areas such as Python, IT, and Networking. The project uses a menu-based system where users can choose which quiz they want to attempt.

---

## Features

- Interactive menu-based quiz system
- Multiple quiz categories:
  - Python Quiz
  - IT Quiz
  - Networking Quiz
- Score tracking for each quiz
- Option to continue or exit after each quiz
- Beginner-friendly Python functions and input handling

---

## Built With

- Python 3

---

## How It Works

1. Run the program
2. The program welcomes the user
3. Choose whether to play or exit
4. Select a quiz category:
   - Python
   - IT
   - Networking
5. Answer the questions
6. Receive your final score
7. Choose to continue or exit
